为什么做币多多(Why):</br>
1.掌握的币讯太少，总是错失先机</br>

2.及时有用的币讯，可以让韭菜变得茁壮~</br>

怎么做币多多(How):</br>

自己做的资讯，不及时，不权威，不全面---->>找有资讯的平台去抓取，哪里去抓？</br>
↓↓↓↓
